🔐 **Forensic Keystroke Dynamics Analyzer**

Инструмент для судебной лингвистической экспертизы, фиксирующий уникальные характеристики клавиатурного почерка:

✅ Задержки между нажатиями клавиш (латентность)
✅ Время удержания клавиш (цифровой аналог силы нажатия)
✅ Диграфы и триграфы с временными метками
✅ Статистический анализ и сравнение образцов
✅ Экспорт в JSON/CSV для судебной экспертизы

🎓 Научный проект.

⚖️ Для установления авторства текстов в судебном процессе.
